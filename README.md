## Versioning

We use [Calendar Versioning](https://calver.org/) for this project. The general format used is `YY.MM.DD.pp` (no leading zeros):

- `YY` is the year
- `MM` is the month
- `DD` is the day
- `pp` is the daily patch

Examples: `24.1.29.0`, `24.1.29.1`
